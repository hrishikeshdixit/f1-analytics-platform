# F1 Analytics Platform 🏎️

An end-to-end data analytics platform for the 2026 F1 season.

## Stack
Python · FastF1 · Google BigQuery · dbt · Looker Studio · Streamlit

## Project Structure
- `ingestion/` — FastF1 data extraction and BigQuery loading
- `dbt/` — Data transformations and quality tests
- `dashboards/` — Looker Studio Dashboard and Streamlit app (in progress)
- `ml/` — Driver fingerprinting model (in progress)

- ✅ Phase 1 — Data Pipeline
- ✅ Phase 2 — dbt Transformations
- ⏳ Phase 3 — Dashboards
- ⏳ Phase 4 — Automation & Containerization
